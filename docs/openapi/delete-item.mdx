# Delete item

<OperationHeader
  serverUrl="https://api-staging.xpayeg.com"
  method="DELETE"
  path="/items/{itemId}"
  hasAuth
  hasPath
/>

Delete a resource by ID.

## Authorization

<div className="openapi-section">

| Name | Type | In | Description |
|------|------|-----|-------------|
| Authorization | Bearer \<token\> | header | JWT or API token |

</div>

## Path Parameters

<div className="openapi-section">

| Name | Type | Required | Description |
|------|------|----------|-------------|
| itemId | string | Yes | The ID of the item to delete |

</div>

## Response Body

<ResponseStatus codes={['204', '404']}>

**204**

Empty response. The resource was deleted successfully.

**404**

```json
{
  "type": "https://example.com/errors/not-found",
  "title": "Not Found",
  "status": 404,
  "detail": "The resource you are trying to access does not exist."
}
```

</ResponseStatus>

  <Tab value="JavaScript">

```javascript
const itemId = 'item_01HXYZ';
const response = await fetch(`https://api-staging.xpayeg.com/items/${itemId}`, {

    Authorization: 'Bearer YOUR_ACCESS_TOKEN',
  },
});
// 204 No Content
```

```python
import requests

item_id = "item_01HXYZ"

    headers={"Authorization": "Bearer YOUR_ACCESS_TOKEN"},
)
# response.status_code == 204
```
