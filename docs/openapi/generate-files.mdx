# generateFiles()

Generate MDX files directly from your OpenAPI schema.

## Usage

Create a script (e.g. `scripts/generate-docs.ts`):

```ts
import { generateFiles } from 'fumadocs-openapi';
import { openapi } from '@/lib/openapi';

void generateFiles({
  input: openapi,
  output: './docs',
  includeDescription: true,
});
```

Run it:

```bash
bun ./scripts/generate-docs.ts
```

## Options

| Option | Description |
|--------|-------------|
| `input` | OpenAPI instance or schema path/URL |
| `output` | Output directory for generated MDX files |
| `includeDescription` | Include operation descriptions in MDX (recommended) |
| `mode` | `'tag'` \| `'file'` \| `'operation'` — one page per tag, file, or operation |
| `groupBy` | `'tag'` \| `'route'` — how to group files into folders |

## Modes

- **Per tag** — one MDX file per OpenAPI tag
- **Per file** — one MDX per schema file (when using multiple schemas)
- **Per operation** — one MDX per operation (e.g. GET /users, POST /users)

Generated pages can use the `<APIPage id="operationId" />` component to render the operation.
