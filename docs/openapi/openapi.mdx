# OpenAPI

Generating docs from your OpenAPI schema with Fumadocs.

- Works in **RSC** environments.
- **OpenAPI 3.0 and 3.1** are supported.

## Manual setup

Install the required packages:

```bash
npm i fumadocs-openapi shiki
```

## Styles

Add the OpenAPI preset to your CSS (e.g. `app/globals.css`):

```css
@import 'fumadocs-openapi/css/preset.css';
```

## Server instance

Create an OpenAPI instance on the server (e.g. `lib/openapi.ts`):

```ts
import { createOpenAPI } from 'fumadocs-openapi/server';

export const openapi = createOpenAPI({
  input: ['./openapi.json'], // or external URL
});
```
