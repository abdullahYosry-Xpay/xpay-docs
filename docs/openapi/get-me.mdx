# Get authenticated user

<OperationHeader
  serverUrl="https://api-staging.xpayeg.com"
  method="GET"
  path="/me"
  hasAuth
/>

Return the current user. Requires Bearer token.

## Authorization

<div className="openapi-section">

| Name | Type | In | Description |
|------|------|-----|-------------|
| Authorization | Bearer \<token\> | header | JWT or API token |

</div>

## Response Body

<ResponseStatus codes={['200', '401', '403']}>

**200**

```json
{
  "id": "usr_01HXYZ",
  "email": "user@example.com",
  "name": "Jane Doe"
}
```

**401**

```json
{
  "error": "unauthorized",
  "message": "Invalid or missing authorization"
}
```

**403**

```json
{
  "type": "https://example.com/errors/forbidden",
  "title": "Forbidden",
  "status": 403,
  "detail": "You are not authorized to access this resource."
}
```

</ResponseStatus>
