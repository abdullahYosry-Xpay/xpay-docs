# Create item

<OperationHeader
  serverUrl="https://api-staging.xpayeg.com"
  method="POST"
  path="/items"
  hasAuth
  hasBody
/>

Create a new resource. Request body required.

## Authorization

<div className="openapi-section">

| Name | Type | In | Description |
|------|------|-----|-------------|
| Authorization | Bearer \<token\> | header | JWT or API token |

</div>

## Request Body

<div className="openapi-section">

| Name | Type | Required | Description |
|------|------|----------|-------------|
| name | string | Yes | Display name |
| amount | integer | No | Amount in smallest currency unit |

Example:

```json
{
  "name": "My Item",
  "amount": 1000
}
```

</div>
