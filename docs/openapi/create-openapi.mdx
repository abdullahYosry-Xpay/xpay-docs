# createOpenAPI

Create an OpenAPI instance on the server for use with [APIPage](/docs/openapi/api-page) and [generateFiles()](/docs/openapi/generate-files).

## Usage

**`lib/openapi.ts`**

```ts
import { createOpenAPI } from 'fumadocs-openapi/server';

export const openapi = createOpenAPI({
  input: ['./openapi.json'],
});
```

## Input

`input` can be:

- **Local path(s):** `['./openapi.json']` or `['./v1.json', './v2.json']`
- **External URL:** `['https://api.example.com/openapi.json']`

## With loader

Optional: add the OpenAPI plugin to your source loader so OpenAPI pages get a badge in the sidebar:

```ts
import { openapiPlugin } from 'fumadocs-openapi/server';
import { loader } from 'fumadocs-core/source';

export const source = loader({
  // ...your source
  plugins: [openapiPlugin()],
});
```

## Notes

- Must run on the **server** (RSC).
- Only **OpenAPI 3.0 and 3.1** are supported.
